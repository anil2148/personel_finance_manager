# Personal Finance Manager

A modern offline-first Android app for tracking income, expenses, budgets, savings goals, recurring payments, and monthly financial progress.

## Features

- Personalized onboarding with income, currency, and financial goal
- Dashboard with balance, monthly totals, remaining budgets, and savings progress
- Add, edit, and delete income or expense transactions
- Category budgets with 80% warnings and overspend alerts
- Savings goals with target dates and progress updates
- Monthly reports with category breakdowns, spending trends, and top categories
- Profile editing, dark mode, PIN lock, reset data, and CSV export placeholder
- Recurring transaction and bill reminder fields
- Premium screen and ad banner placeholders ready for future monetization

## Tech Stack

- Kotlin and Jetpack Compose
- Material 3
- MVVM with Repository pattern
- Room database with Kotlin coroutines and Flow
- Gradle Kotlin DSL
- Minimum SDK 26

## Run

1. Open the project in Android Studio Ladybug or newer.
2. Allow Gradle sync to complete.
3. Select an emulator running Android 8.0 (API 26) or newer.
4. Run the `app` configuration.
